funkyheatmappy.position_arguments.position_arguments(row_height=1, row_space=0.1, row_bigspace=0.5, col_width=1, col_space=0.1, col_bigspace=0.5, col_annot_offset=3, col_annot_angle=30, expand_xmin=0, expand_xmax=2, expand_ymin=0, expand_ymax=0)¶

Defines parameters for positioning in a plot.

Parameters:

  • row_height (float) – The height of the rows.

  • row_space (float) – The space between rows.

  • row_bigspace (float) – The large space between row groups.

  • col_width (float) – The width of the columns.

  • col_space (float) – The space between columns.

  • col_bigspace (float) – The large space between column groups.

  • col_annot_offset (float) – How much the column annotation will be offset by.

  • col_annot_angle (float) – The angle of the column annotation labels.

  • expand_xmin (float) – The minimum expansion of the plot in the x direction.

  • expand_xmax (float) – The maximum expansion of the plot in the x direction.

  • expand_ymin (float) – The minimum expansion of the plot in the y direction.

  • expand_ymax (float) – The maximum expansion of the plot in the y direction.
